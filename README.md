# Unity Project Cleaner

ホームディレクトリを丸ごとスキャンして、Unity プロジェクトを一覧表示・整理できるターミナルツールです。

## 必要環境

- macOS
- Python 3.9 以上
- [rich](https://github.com/Textualize/rich) ライブラリ

## インストール

```bash
pip3 install rich
```

## 使い方

```bash
python3 unity_cleaner.py
```

## 機能

### プロジェクト一覧
- ホーム以下を自動スキャンして Unity プロジェクトを検出
- スキャン対象フォルダは起動時に追加・削除できる
- プロジェクト名・Unity バージョン・最終更新日・サイズ・Git 有無・Unity Hub 登録状態・メモを一覧表示
- 同名プロジェクトが複数ある場合は ⚠ で警告

### 削除スコア (自動判定)
各プロジェクトを以下の基準でスコアリングし、削除候補を自動判定します。

| 条件 | スコア |
|------|--------|
| 1年以上更新なし | +3 |
| 半年以上更新なし | +2 |
| 3ヶ月以上更新なし | +1 |
| 実サイズ 10MB 未満 | +2 |
| シーンが 2個以下 | +2 |
| Git 管理なし | +1 |
| 名前に test/tmp/sample 等を含む | +1 |

- 🔴 5点以上 → 削除候補
- 🟡 3〜4点 → 要確認
- 🟢 2点以下 → 保持

### 詳細表示
番号を選ぶと以下の詳細を確認できます。

- Unity バージョン・更新日・サイズ・Library サイズ・Git 管理状況
- シーン一覧
- C# スクリプト数
- アセット種別ごとの数（上位 10件）
- メモ

### ソート
以下の順で一覧を並び替えできます。

- 削除スコア順
- 更新日（古い・新しい順）
- サイズ（大きい・小さい順）
- 名前順

### メモ
各プロジェクトにメモを残せます。  
メモはプロジェクトフォルダ内の `.unity_memo` ファイルに保存されます（フォルダを移動してもメモが一緒についてきます）。

### Unity Hub 連携
- 一覧で Hub 登録状態を確認
- 未登録プロジェクトを選択して Hub に一括登録

### ゴミ箱に移動
不要なプロジェクトをゴミ箱に移動します（完全削除ではないので復元可能）。

### Library キャッシュクリア
プロジェクト自体は残しつつ `Library` フォルダだけ削除してディスクを節約できます。  
Unity で開くと自動的に再生成されます。

## Unity プロジェクトの判定方法

以下のファイルが存在するフォルダを Unity プロジェクトとして認識します。

```
ProjectSettings/ProjectVersion.txt
```
